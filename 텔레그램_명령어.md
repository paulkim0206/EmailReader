# 🤖 지능형 이메일 비서 - 텔레그램 명령어 시스템 기획서

사용자님의 훌륭한 아이디어를 실현하기 위해 텔레그램 명령어(Command) 기능들을 하나씩 모아두는 곳입니다!
여기에 적어두신 아이디어는 언제든지 비서 봇 파이썬 코드에 추가될 수 있습니다.

## 📌 현재 구상 및 적용된 명령어 리스트
1. **`/save` (또는 인라인 버튼 💾)**
   - **기능**: 요약된 메일 정보와 원문을 내 컴퓨터의 로컬 경로(`Email_Reports` 폴더)에 깔끔한 워드 문서(.docx)로 다이렉트 자동 저장합니다.
   - **상태**: 현재 인라인 버튼(메시지 하단 클릭 방식)으로 완벽하게 구현되어 작동 중입니다. 앞으로도 명령어로서 보존됩니다.

2. **`/draft` (명령어 아이디어 - 진행 대기중!)**
   - **기능**: 단순히 요약을 넘어, 해당 메일에 바로 답장할 수 있는 '최고급 무역/영업 회신 초안'을 작성해 달라고 AI에게 특별히 부탁합니다.
   - **설명**: 메일이 올 때마다 무조건 초안을 쓰게 해서 봇 메시지 알림이 불필요하게 길어지거나 성능이 낭비되는 것을 막고, 꼭 필요한 메일에만 사용자가 명령하여 초안을 요청하는 고품격 기능입니다.
   - **상태**: 신규 아이디어 등록 완료 (개발 전)
