# 🚀 지능형 이메일 비서 (Email Assistant) - 패치노트 및 버전 관리

이 문서는 시스템 설계자와 개발자가 어떻게 아이디어를 주고받으며 우리 비서를 진화시켜왔는지 한눈에 볼 수 있는 **개발 일지(Release Notes)** 문서입니다.

---

## 🟢 [v1.5.1] - 문서 자동화 엔진 교체 및 텔레그램 컨트롤러 탑재 (현재 최신버전!)
- **마크다운(.md) 최적화 저장 엔진**: 기존 워드(.docx) 파일 저장 시 발생하던 한글 깨짐 및 레이아웃 붕괴 현상을 완벽히 해결하기 위해, 에러율이 0%인 순수 파이썬 마크다운(Markdown) 문서 저장 방식으로 `local_storage.py` 구조를 100% 교체 완료했습니다.
- **텔레그램 상태 확인 리모컨**: 언제 어디서든 스마트폰으로 `/status` 명령어를 입력하면 비서 봇이 즉각 "정상 생존" 상태를 든든하게 보고하도록 양방향 통신망을 구축했습니다.
- **AI 보고서 초경량화 다이어트**: 스마트폰의 좁은 화면을 너무 많이 가리던 불필요한 '실무 조언' 등 군더더기 텍스트를 AI 뇌파 교육 단계에서 원천 차단하여 가독성을 한계까지 끌어올렸습니다.
- **불필요한 명령어 컷오프**: 무의미한 `/save` 타자 타이핑 기능은 도려내고, 오직 원클릭 저장 버튼 하나만으로 깔끔하게 작동하게 만들어 시스템 복잡성을 과감히 덜어냈습니다.

## 🔵 [v1.4.0] - 데스크톱 GUI 관리자 창 도입 및 독립 실행형 변신
- **데스크톱 앱(exe) 변신**: 칙칙하고 텍스트만 뜨던 구조를 버리고, 바탕화면에 띄워두고 쓰는 디자인된 **'조종석(GUI) 패널'**을 개발했습니다. 사용자 요청을 반영하여 황금 비율(400x650)의 세로형 모던 다크모드를 강제 적용했습니다.
- **실행 파일(.exe) 패키징 로직**: 코딩 프로그램 없이, 바탕화면에서 아이콘을 더블클릭하는 것만으로 GUI와 엔진이 통신하도록 내부 아키텍처를 전면 개조했습니다 (`--daemon` 백그라운드 라우팅 기법).
- **종료 버튼 강제제어 버그 완벽 개선**: 윈도우 OS 특유의 백그라운드 좀비 현상을 파악하여, 사용자 명령 인가 시 시스템 핵심 권한(`taskkill`)으로 엔진의 숨통을 확실히 끊도록 타격 범위를 상향했습니다.

## 🔵 [v1.3.0] - AI 디테일 스펙업 및 가시성 극대화
- **가시성 최적화 (사용자 아이디어 노트)**: 텔레그램의 마크다운 문자(`*`, `-`) 충돌 오류를 발견하여 통신망을 HTML 태그 방식으로 완전히 뜯어고쳤습니다. AI 요약 시 여백(엔터)과 단정한 글머리 기호(•)를 강제하게 하여 스마트폰 가독성을 가장 높은 수준으로 끌어올렸습니다!
- **과거 메일 스킵 (사용자 아이디어 노트)**: 서버 전원을 처음 켰을 때 과거에 쌓인 수십 통의 안 읽은 메일이 알람 테러를 보내는 현상을 막기 위해, 시동 직전에 감지된 옛날 메일은 기계적으로 조용히 무시(침묵)하고 새 메일만 받아오도록 스킵 로직을 적용했습니다.

## 🟡 [v1.2.0] - 메일 보고서 자동화 및 알림 최적화
- **워드 보고서 다이렉트 저장**: 단순 텍스트 알람을 넘어, 사용자가 원할 때 텔레그램에서 추가 버튼을 누르면 AI의 분석 결과를 즉시 깔끔한 워드 문서(`.docx`) 파일로 가공하여 내 컴퓨터 방어 금고(`Email_Reports` 폴더)에 자동 저장하는 혁신적 기능을 심어두었습니다.
- **비대화형 기능 다이어트**: 메시지 알림이 불필요하게 무거워지는 것을 막기 위해, 매번 강제 생성되던 '회신 초안' 과정을 과감히 백지화하고, 사용자가 직접 명령을 내릴 때만 통제할 수 있도록 시스템을 가볍게 다이어트했습니다. (명령어 대기 문서 생성 완료)

## 🟠 [v1.1.0] - 메인 통신 연결망 교체 및 동보체인 그룹웨어 해킹
- **구글 AI 두뇌 코어 최신화**: 기존의 낡은 API를 뜯어내고, 구글이 가장 최근 발표한 최신형 공식 문법 패키지(`google-genai`)와 **'gemini-3-flash-preview'**(제미나이 3.0!) 최상위 두뇌 모델로 칩을 성공적으로 교체 이식했습니다.
- **그룹웨어 암호화 통신망 해독 완수**: 보편적인 이메일 서버 구조와 다른 현재 그룹웨어만의 강력한 보안 통신 스펙을 파훼하고, 직접 UID(메일 고유등록번호)를 메인 서버에 찔러 넣는 특수 통신 방식으로 원본 이메일을 완벽히 훼손 없이 꺼내오는데 성공했습니다.

## 🔴 [v1.0.0] - 프로젝트 초기 모델 탄생
- 사용자의 동보체인 이메일 계정을 백그라운드 환경에서 정기적으로(1분) 순찰하고 통신하는 기본형 스케줄러 뼈대가 작동하기 시작했습니다.
